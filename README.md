# Glowave Creative Studio

A boutique creative agency landing page built with **React**, **Tailwind CSS**, and **Framer Motion**. Optimized for high-end UI/UX and smooth digital experiences.

---
## 🔗 Live Demo

Experience the live version of Glowave Creative Studio here:  
👉 **[Live Preview on Vercel](https://glowave-agency-portal.vercel.app/)**


## 💎 Features
* **Minimalist UI:** Modern Dribbble-inspired design.
* **Dual Language:** Full support for English and Bengali.
* **Smooth Motion:** Scroll-triggered animations and seamless transitions.
* **Responsive:** Pixel-perfect layout for all devices.
* **Clean Code:** Modular component-based architecture.

## 🛠️ Tech Stack
* **Core:** React.js
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React
* **Routing:** React Router

## 🚀 Installation

```bash
# Clone the repository
git clone [https://github.com/yourusername/glowave-agency.git](https://github.com/yourusername/glowave-agency.git)

# Install dependencies
npm install

# Run locally
npm run dev
